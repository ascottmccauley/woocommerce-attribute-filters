== Description ==
Automatically adds filters for all available products in the current category. Similar to adding all possible Layered Nav Widgets would do.


== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "WooCommerce Attribute Filters".
3. Click install.
4. Click activate.
